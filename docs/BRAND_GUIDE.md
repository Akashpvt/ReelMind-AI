# Brand Guide

## Brand Personality

ReelMind AI should feel premium, fast, futuristic, and creator-focused. It should communicate creative confidence without feeling cold or overly technical.

## Visual Direction

The product should use a dark premium UI with purple glow, blue neon highlights, and glassmorphism-inspired surfaces.

## Logo Concept

Create a futuristic wordmark for **ReelMind AI** with an abstract icon combining:

- A reel or play-button shape
- A neural spark or AI node pattern
- A subtle circular motion cue

The icon should work as:

- App icon
- Favicon
- Social avatar
- Loading mark

Logo mood:

- Premium
- Minimal
- High contrast
- Neon-accented
- Easy to recognize at small sizes

## 3D Animation Direction

Use 3D motion sparingly to create a premium AI-product feel.

Suggested elements:

- Floating glass panels representing generated content
- A glowing AI core or orb
- Subtle reel rings orbiting around a central idea node
- Prompt cards moving through a creative pipeline
- Purple and blue light trails

Tools:

- Spline for fast hero scene exploration
- Three.js for custom interactive scenes
- React Three Fiber for React integration

Motion style:

- Smooth
- Calm
- Premium
- Responsive to cursor movement
- No distracting loops in productivity areas

## UI Moodboard Description

The UI should feel like a premium creator command center:

- Dark layered backgrounds
- Glass panels with subtle blur
- Purple-to-blue neon accents
- Sharp typography
- Compact dashboard cards
- Smooth Framer Motion transitions
- High-contrast buttons
- Clear empty states
- Soft glow around primary actions

Avoid:

- Overly playful colors
- Busy gradients
- Low-contrast text
- Excessive animation
- Generic SaaS illustrations
