# MVP Scope

## Phase 1 MVP Features

Only the following features are included in the Phase 1 MVP.

## 1. Viral Hook Generator

Generate short, punchy hook options for reels based on:

- Topic
- Niche
- Audience
- Content goal
- Tone

Expected output:

- 5 to 10 hook options
- Optional hook style labels such as curiosity, contrarian, pain-point, transformation, or list-based

## 2. Reel Script Generator

Generate a short-form reel script with a clear structure:

- Hook
- Opening context
- Main points
- Payoff
- CTA

Expected output:

- 15-second script
- 30-second script
- Optional voiceover-friendly version

## 3. Caption Generator

Generate platform-ready captions that support the reel.

Expected output:

- Short caption
- Long caption
- Hashtag suggestions
- Optional emoji-free variant

## 4. CTA Generator

Generate clear calls to action for different creator goals.

CTA goals may include:

- Follow
- Comment
- Save
- Share
- Visit link
- Buy product
- Book service

## 5. Creator Dashboard

Provide a simple home base for users.

Dashboard should show:

- Recent projects
- Recently generated prompts
- Quick action buttons
- Basic account state

## 6. Prompt History

Store previous prompts and generated outputs so creators can revisit work.

History should include:

- Original prompt
- Output type
- Date created
- Saved project relation if applicable

## 7. Project Save Option

Allow users to save generated content into projects.

Project data should include:

- Project title
- Niche or topic
- Hook
- Script
- Caption
- CTA
- Created date
- Updated date

## Out of Scope for Phase 1

The following are intentionally excluded from Phase 1:

- Full video generation
- Full image generation
- Team workspaces
- Billing
- Advanced analytics
- Social media scheduling
- Native mobile app
- Browser extension
- Agency client portals
- Public template marketplace

## MVP Boundary Rule

Phase 1 should focus on text-based content generation, saving, and retrieval. Any feature that requires complex media generation, multi-user collaboration, or monetization infrastructure should be deferred.
